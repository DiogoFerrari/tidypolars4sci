** Write to file

To write the tibble to a file, use ~save_data()~. The type to be used is inferred from the filename extension.  Relative paths are acceptable in the filename string ~fn~, which receives the filename and its path. These are the file types available to save the ~tibble~ to file:

# 
#+BEGIN_SRC python :exports results :results output code org :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
import tidypolars4sci as tp
tp.tibble_df.__get_accepted_output_formats__(True)
#+END_SRC

#+RESULTS[0fc2f310cd437683a9ffe062093eedce886cea3b]:
#+begin_src org
- csv-like: csv, dat, tsv, txt
- excel-like: ods, xls, xlsx, xlt, xltx
- latex: tex
- Stata files: dta
- parquet: parquet
#+end_src

** Examples

*** Save to .csv

Here is an example of how to save a ~tibble~ to ~.csv~. We can use the mtcars data provided with tidypolars\(^{4sci} \) to illustrate the procedure:

#+BEGIN_SRC python :exports both :results output code :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
import tidypolars4sci as tp
from tidypolars4sci.data import mtcars as df

df.head().print()
#+END_SRC

To save that ~tibble~ to a ~csv~ file use:

# 
#+BEGIN_SRC python :exports both :results output code :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
# save data to csv in the folder Documents
df.save_data(fn = '~/Documents/mtcars-data.csv')

# Relative paths are accepted: save data to csv in the current working folder
# df.save_data(fn = './mtcars-data.csv')
#+END_SRC

*** Saving copies
:PROPERTIES:
:CUSTOM_ID: saving-copies
:END:

It is possible to save copies in other formats without calling the function multiple times using the argument ~copies~. For instance, to save a ~.csv~ file, a copy in ~.xlsx~, and another in ~.dta~, use:

# 
#+BEGIN_SRC python :exports both :results output code :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
import tidypolars4sci as tp
from tidypolars4sci.data import mtcars as df

# save data to csv
df.save_data(fn = '~/Documents/mtcars-data.csv', copies = ["xlsx", ".dta"])
#+END_SRC

*** Saving to LaTeX

tidypolars\(^{4sci} \) has a function ~to_latex()~ that exports a ~tibble~ to LaTeX. Check documentation [[file:../../api/#tidypolars4sci.tibble_df.tibble.to_latex][here]].



It is possible to export directly from ~save_data~ using either as a copy, using the argument ~copies~ (see [[#saving-copies][Saving copes]]) or using a filename with ~.tex~ extension.

1. Using a filename with ~.tex~ extension:

# 
#+BEGIN_SRC python :exports both :results output code :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
from tidypolars4sci.data import mtcars as df

df.save_data(fn='~/Documents/table.tex')
#+END_SRC

#+RESULTS[217eb7e2c071661912c099f2d3d7e7be00feb1f5]:
#+begin_src python
Saving table.tex...done!
Save at: ~/Documents
#+end_src


1. Using the ~copies~ argument:

#+BEGIN_SRC python :exports both :results output code :tangle src-write.py :cache yes :noweb no :session *Python* :linenums 1 :eval always
from tidypolars4sci.data import mtcars as df

df.save_data(fn='~/Documents/table.csv', copies = ['tex'])
#+END_SRC

#+RESULTS[020838d26b078e272fd695f8d9eb3b0c60b3f8c8]:
#+begin_src python
Saving table.tex...done!
Saving table.csv...done!
Save at: ~/Documents
#+end_src

The latex table in the ~.tex~ file will look like this (for customization options, see [[file:../../api/#tidypolars4sci.tibble_df.tibble.to_latex][here]]):

# table: mtcars
#+BEGIN_SRC python :exports results :results output code org :tangle src-write.py :cache yes :noweb no :session *Python*
print(df.to_latex())
#+END_SRC
